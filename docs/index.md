# CCMS Portal Documentation

Welcome to the CCMS Portal documentation.

## Project Structure
- **Web Portal (`ccms/`)**: A Next.js application for the console.
- **Hardware (`ESP32_wifi_CCMS_v1.0/`, `v2.0/`)**: C++ firmware based on PlatformIO to interact with mqtt and AWS.
- **Backend (`Lambda/`)**: Serverless Python AWS Lambda functions.
